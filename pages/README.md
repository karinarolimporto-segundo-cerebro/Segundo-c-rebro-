# 🧠 Segundo Cérebro Premium

Dashboard sincronizado para iPhone, iPad e Windows com Firebase.

## ✨ Funcionalidades

- 📅 **Calendário Inteligente** - Vista mensal, semanal e diária com cores
- 💊 **Medicamentos com IA** - Foto da caixa → Extrai automaticamente
- 💰 **Finanças Familiares** - Controle de despesas com marido e mãe
- 📁 **Pasta de Modelos** - Salva links e inspirações da web

## 🚀 Como instalar e usar

### 1. No seu computador (Windows):

1. Abre o terminal ou prompt de comando
2. Instala dependências:
   ```
   npm install
   ```

3. Inicia o servidor de desenvolvimento:
   ```
   npm run dev
   ```

4. Abre `http://localhost:3000` no navegador

### 2. Para fazer deploy no Vercel:

1. Você já tem a conta Vercel criada
2. Vercel vai detectar automaticamente quando você conectar o GitHub
3. Pronto! Funciona em qualquer lugar

## 🔐 Firebase

O app já vem configurado com Firebase. Os dados sincronizam automaticamente!

## 📱 Como acessar no iPhone/iPad

1. Abre Safari
2. Vai no link que você receber do Vercel
3. Clica em "Compartilhar" → "Adicionar à Tela Inicial"
4. Pronto! Fica como um app
